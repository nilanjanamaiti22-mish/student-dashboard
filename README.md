# 📊 Student Performance Dashboard

An interactive web dashboard built with Streamlit to analyze and visualize 
student exam performance — built as a tool for teachers to quickly identify 
trends, at-risk students, and grade distribution.

## Features
- Filter students by section and minimum marks
- View class average, highest score, pass rate, and at-risk count
- Visualize marks distribution (bar chart) and grade distribution (pie chart)
- Sortable, searchable student data table

## Tech Stack
- Python
- Pandas (data handling)
- Streamlit (web app framework)
- Plotly (interactive charts)

## How to run locally
```bash
pip install streamlit pandas plotly
streamlit run app.py
```
